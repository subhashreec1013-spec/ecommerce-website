# 🛒 E-Commerce Website

A responsive multi-page e-commerce website developed during my internship at **iStudio** using HTML, CSS, and JavaScript.

The project demonstrates the design and development of a basic online shopping platform with product browsing, cart functionality, and checkout workflow.

---

## 🌐 Live Demo

https://subhashreec1013-spec.github.io/ecommerce-website/

---

## 📌 Project Overview

This project simulates a simple online shopping experience where users can browse products, view product details, add items to cart, and proceed through the checkout process.

The website focuses on creating a clean UI and structured navigation similar to real-world e-commerce platforms.

---

## 🚀 Features

- Product listing page
- Product detail pages
- Category browsing
- Shopping cart page
- Checkout process
- Order confirmation page
- Search functionality
- Responsive layout

---

## 🛠 Technologies Used

### Design
- Responsive UI design
- Clean product layout
